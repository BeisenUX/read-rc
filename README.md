## 描述
提供给 storybook-lib、bscpm 模块中用于获取组件项目中配置信息

### API
#### RCFileName 获取配置文件的名字

#### getPackageInfo() 是否存在package.json获取都不会报错

#### extractRCFromPakcage() 从package.json文件中提取配置信息

#### default() 获取配置文件json格式数据
